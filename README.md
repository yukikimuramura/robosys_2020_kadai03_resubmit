# robosys_2020_kadai03_resubmit
### 概要
1~100までのフィナボッチ数列を並べてみました.
### 動作環境
- Rasberry Pi Model 3B+
- Ubuntu18.04
### 実行手順
```
1.リポジトリをクローンする.
$ cd ~/catkin_ws/src
2.ディレクトリに入る.
$ git clone https://github.com/yukikimuramura/mypkg.git
3.コンパイルする
$ gcc -o kadai3 kadai3.c
4.実行する
$ ./kadai3
```
